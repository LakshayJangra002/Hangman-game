# 🎮 Hangman Game in Python

A simple text-based Hangman Game built using Python.

## 📖 About the Project

This is a beginner-friendly Hangman game that runs in the terminal. The player guesses one letter at a time to reveal the hidden word. The player has a maximum of **6 incorrect guesses** before the game ends.

## ✨ Features

- Randomly selects a word from a predefined list
- 5 predefined words
- Maximum 6 incorrect guesses
- Input validation
- Prevents duplicate guesses
- Win and Lose conditions
- Beginner-friendly Python code

1. Install Python 3.

3. Open a terminal in the project folder.
4. Run:

```bash
python hangman.py
```
